Cybersecurity Hierarchy

Microsoft, as well as other organizations post information about tracked Lebanon Advanced Persistent Threats. Below are the APTs and their Tactics, Techniques, and Procedures.


Polonium-Plaid Rain

This Group is believed to have been active since September 2021. MSTIC (Microsoft Threat Intelligence) states that they are highly confident this group is based in Lebanon. While Polonium has not been given an official APT number, this group is still big enough to be being tracked by multiple threat intelligence teams for its activities. MSTIC also states that they are moderate confidence that Polonium coordinates operations with multiple tracked Iranian Threat Actors based on victimm overlap, as well as common attack vectors and tools.

Targets

Poloniums targets include many different organizations within Israel.

Attack Vectors (Arghire, 2023)

  • CreepyDrive-Powershell Backdoor - supports Command Execution
  • CreepySnail-Powershell Backdoor - supports Command Execution
  • DeepCreep-C# Backdoor - can retrieve commands from text files on Dropbox - can upload/download files to/from the cloud service - achieves persistance by placing a shortcut file in the Startup folder and by creating a scheduled task
  • MegaCreep-Retrieves commands from text files stored in Mega cloud storage - appears to be newer version of DeepCreep, reusing some of its code
  • FlipCreep-C# Backdoor that reads commands from a text file on an FTP server
  • TechnoCreep-Relies on TCP sockets for C&C communication
  • PapaCreep-C++ Backdoor that uses different components to read commands from a file, to communicate with the C&C server, to upload files to the C&C server, and to download files from the server

Volatile Cedar-Lebanese Cedar-Caterpillar-Explosive RAT Group

Volatile Cedar is a Lebanese cyber-espionage group active since 2012, targeting individuals, companies, and institutions worldwide. The group's activities are driven by political and ideological objectives, aligning with broader regional agendas.

Targets

Defense Contractors, Media Companies, Telecomunications Organizations, and Educational Institutions in Israel, US, UK, Lebanon, and Egypt.

Attack Vectors (Volatile CeDAr, Lebanese Cedar, Group G0123 | MITRE ATT&CK®, n.d.)

  • Active Scanning: Vulnerability scanning-Scans of target server
  • Active Scanning: Wordlist Scanning-used DirBuster and GoBuster to bruteforce web directories and DNS subdomains
  • Exploit Public-Facing Application-targeted publicly facing web servers, with both automatic and manual vulnerability discovery.
  • Ingress Tool Transfer-can deploy additional tools.
  • Server Software Component: Web Shell-an inject web shell code into a server.

Dark Caracal-BlackOasis-ZooPark-LGSD

Dark Caracal is a threat group linked to the Lebanese General Directorate of General Security (GDGS) and has been active since at least 2012. The group is involved in cyber-espionage campaigns targeting a range of global victims, often focusing on government and military sectors.

Targets

Military Personnel, Activists, Government Officials, IT Institutions, Defense Contractors, Healthcare- Germany, France, Middle Eastern Countries.

Attack Vectors (Dark Caracal, Group G0070 | MITRE ATT&CK®, n.d.)

  • Application Layer Protocol: Web Protocols-Dark Caracal's version of Bandook communicates with their server over a TCP port using HTTP payloads Base64 encoded and suffixed with the string "&&&".
  • Boot or Logon Autostart Execution: Registry Run Keys / Startup Folder-adds a registry key to HKEY_USERS\Software\Microsoft\Windows\CurrentVersion\Run for persistence
  • Command and Scripting Interpreter: Windows Command Shell
  • Data from Local System-Collected data from files
  • Drive-by Compromise
  • File and Directory Discovery
  • Obfuscated Files or Information: Software Packing
  • Obfuscated Files or Information: Encrypted/Encoded File
  • Phishing: Spearphishing via Service
  • Screen Capture-using Malware
  • System Binary Proxy Execution: Compiled HTML File
  • User Execution: Malicious File-makes their malware look like Flash Player, Office, or PDF documents in order to entice a user to click on it
  • Application Layer Protocol: Web Protocols-implants using standard HTTP communication
