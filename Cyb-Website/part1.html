Part 1 - Nation-State Infrastructure

Cybersecurity Hierarchy

Microsoft, as well as other organizations post information about tracked Lebanon Advanced Persistent Threats. Below are the APTs and their Tactics, Techniques, and Procedures.


Polonium-Plaid Rain

This Group is believed to have been active since September 2021. MSTIC (Microsoft Threat Intelligence) states that they are highly confident this group is based in Lebanon. While Polonium has not been given an official APT number, this group is still big enough to be being tracked by multiple threat intelligence teams for its activities. MSTIC also states that they are moderate confidence that Polonium coordinates operations with multiple tracked Iranian Threat Actors based on victimm overlap, as well as common attack vectors and tools.

Targets

Poloniums targets include many different organizations within Israel.

Attack Vectors (Arghire, 2023)

  • CreepyDrive-Powershell Backdoor - supports Command Execution
  • CreepySnail-Powershell Backdoor - supports Command Execution
  • DeepCreep-C# Backdoor - can retrieve commands from text files on Dropbox - can upload/download files to/from the cloud service - achieves persistance by placing a shortcut file in the Startup folder and by creating a scheduled task
  • MegaCreep-Retrieves commands from text files stored in Mega cloud storage - appears to be newer version of DeepCreep, reusing some of its code
  • FlipCreep-C# Backdoor that reads commands from a text file on an FTP server
  • TechnoCreep-Relies on TCP sockets for C&C communication
  • PapaCreep-C++ Backdoor that uses different components to read commands from a file, to communicate with the C&C server, to upload files to the C&C server, and to download files from the server

Volatile Cedar-Lebanese Cedar-Caterpillar-Explosive RAT Group

Volatile Cedar is a Lebanese cyber-espionage group active since 2012, targeting individuals, companies, and institutions worldwide. The group's activities are driven by political and ideological objectives, aligning with broader regional agendas.

Targets

Defense Contractors, Media Companies, Telecomunications Organizations, and Educational Institutions in Israel, US, UK, Lebanon, and Egypt.

Attack Vectors (Volatile CeDAr, Lebanese Cedar, Group G0123 | MITRE ATT&CK®, n.d.)

  • Active Scanning: Vulnerability scanning-Scans of target server
  • Active Scanning: Wordlist Scanning-used DirBuster and GoBuster to bruteforce web directories and DNS subdomains
  • Exploit Public-Facing Application-targeted publicly facing web servers, with both automatic and manual vulnerability discovery.
  • Ingress Tool Transfer-can deploy additional tools.
  • Server Software Component: Web Shell-an inject web shell code into a server.

Dark Caracal-BlackOasis-ZooPark-LGSD

Dark Caracal is a threat group linked to the Lebanese General Directorate of General Security (GDGS) and has been active since at least 2012. The group is involved in cyber-espionage campaigns targeting a range of global victims, often focusing on government and military sectors.

Targets

Military Personnel, Activists, Government Officials, IT Institutions, Defense Contractors, Healthcare- Germany, France, Middle Eastern Countries.

Attack Vectors (Dark Caracal, Group G0070 | MITRE ATT&CK®, n.d.)

  • Application Layer Protocol: Web Protocols-Dark Caracal's version of Bandook communicates with their server over a TCP port using HTTP payloads Base64 encoded and suffixed with the string "&&&".
  • Boot or Logon Autostart Execution: Registry Run Keys / Startup Folder-adds a registry key to HKEY_USERS\Software\Microsoft\Windows\CurrentVersion\Run for persistence
  • Command and Scripting Interpreter: Windows Command Shell
  • Data from Local System-Collected data from files
  • Drive-by Compromise
  • File and Directory Discovery
  • Obfuscated Files or Information: Software Packing
  • Obfuscated Files or Information: Encrypted/Encoded File
  • Phishing: Spearphishing via Service
  • Screen Capture-using Malware
  • System Binary Proxy Execution: Compiled HTML File
  • User Execution: Malicious File-makes their malware look like Flash Player, Office, or PDF documents in order to entice a user to click on it
  • Application Layer Protocol: Web Protocols-implants using standard HTTP communication

Mission Statement

Lebanon's Cyber Warfare department strengthens law enforcements in reducing the impact of cyber threats. They provide relevant smart data and threat reports using an innovative reliable framework. (Lebanon CERT, 2024)

5 Critical Objectives

  1. Developing Strong Relationships with Police and Military Law Enforcement units
  2. Developing Strong Cyber Warfare Skills among Military Members through Higher Education
  3. Aggregating and Analyzing Threat Data
  4. Innovating Threat Intelligence to become a Leader in the Threat Intelligence World
  5. Sharing and Acting Upon Creative, Relevant, and Strong Internally Created Threat Reports and Dashboards

Operatives Section

Lebanon's Cyber Team

Head of Cyberwarfare

Lebanon's head of cyberwarfare remains unknown largely due to the country's relatively small size and limited resources in the realm of cyberspace. Unlike larger countries with more detailed cyber infrastructure and better established cybersecurity frameworks, Lebanon's cyber capabilities are still developing. Because of this, the knowledge of who the head of Cyberwarfare is able to remain unknown to the public.

Military-Government Cyber Subordinates

President of Lebanon Michel Aoun

As the head of state, the president is the supreme commander of the Lebanese Armed Forces and ultimately oversees national defense and security. Although not directly involved in cyber operations, the president’s leadership influences national policies, including cyber defense strategy.

Prime Minister of Lebanon Najib Mikati

The Prime Minister heads the executive branch of the government. He has oversight of the Ministry of Defense and the Ministry of Interior and Municipalities, both of which contain divisions responsible for national security, including cybersecurity.

Minister of Defense Maurice Slim

The Ministry of Defense is responsible for national defense, which includes military cybersecurity initiatives. The Ministry oversees the armed forces, where the Military Intelligence Directorate plays a role in cyber warfare and defense strategies

Commander of Lebanese Armed Forces General Joseph Aoun

The commander leads the entire military, including the intelligence and cybersecurity units. Under his command, the Military Intelligence Directorate plays a key role in protecting military networks and critical infrastructure from cyber threats.

Military Intelligence Directorate Brigadier General Wissam Safi

This directorate is responsible for intelligence gathering, including in cyberspace. It oversees cybersecurity operations within the military, focusing on the protection of military systems, counter-cyberespionage, and ensuring the integrity of communication systems.

Minister of Interior and Municipalities

The General Directorate of General Security oversees national security matters, including surveillance and counterintelligence, which increasingly involve cyberspace activities. The General Security agency would be responsible for monitoring and addressing cyber threats that impact national security

Deputy Director-General

The Deputy Director-General oversees critical divisions that deal with cybersecurity. These divisions include counterintelligence and the cybersecurity division, both key in combating cyber threats to the nation.

APTs That Have Attacked Lebanon:

Leafminer

Leafminer is an Iranian threat group that has targeted government organizations and business entities in the Middle East since at least early 2017. Listed below are some of their Attack Vectors.

  • Brute Force: Password Spraying
  • Command and Scripting Interpreter: JavaScript
  • Credentials from Password Stores and Web Browsers
  • Drive-by Compromise
  • Email Collection: Remote Email Collection
  • OS Credential Dumping: LSASS Memory

Molerats

Molerats is an Arabic-speaking, politically-motivated threat group that has been operating since 2012. The group's victims have primarily been in the Middle East, Europe, and the United States. Listed below are some of their Attack Vectors.

  • Command and Scripting Interpreter: PowerShell
  • Ingress Tool Transfer
  • Phishing: Spearphishing Attachment
  • Process Discovery
  • User Execution: Malicious Link
  • User Execution: Malicious File

MuddyWater

MuddyWater is a cyber espionage group assessed to be a subordinate element within Iran's Ministry of Intelligence and Security (MOIS).[1] Since at least 2017, MuddyWater has targeted a range of government and private organizations across sectors, including telecommunications, local government, defense, and oil and natural gas organizations, in the Middle East, Asia, Africa, Europe, and North America. Listed below are some of their Attack Vectors.

  • Abuse Elevation Control Mechanism: Bypass User Account Control
  • Boot or Logon Autostart Execution: Registry Run Keys / Startup Folderr
  • Phishing: Command and Scripting Interpreter: Windows Command Shell
  • Credentials from Password Stores and Web Browsers
  • Data Encoding: Standard Encoding
  • Exploitation of Remote Services
